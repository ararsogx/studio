# **App Name**: Learniverse Hub

## Core Features:

- User Authentication and Roles: Secure user registration and login with role-based access control (Standard, Premium, Admin).
- Learning Materials Management: Upload, manage, and categorize learning resources (modules, notes, exams) in various formats (PDF, PPT, DOCX).
- AI-Powered Study Assistant: Integrated AI tool to answer student's academic questions with a limited number of free questions per day for standard users.
- Offline Resource Access: Enable secure offline access to downloaded resources within the platform, preventing external sharing and local downloads. Protection via DRM will require a cloud service.
- Grade Calculator: A grade calculator where students can track their academic progress using this built-in tool.
- Admin Payment Approval: Admin dashboard for managing user subscriptions, approving premium user payments, and managing user roles.
- Firestore Database Integration: Database structure for securely storing user data, learning materials, progress tracking, and system configurations.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for a professional and trustworthy feel.
- Background color: Light gray (#F0F2F5), creating a clean and modern backdrop.
- Accent color: A vibrant orange (#FF9800) for call-to-action buttons and important notifications to guide user actions.
- Body font: 'Inter' (sans-serif) for a modern, neutral, readable interface.
- Headline font: 'Space Grotesk' (sans-serif) for a computerized, techy, scientific feel
- Use a consistent set of minimalist icons from a library like FontAwesome or Material Icons.
- Design a responsive layout with clear content hierarchy and intuitive navigation.
